# Buy the Dip Strategy Backtest

This repository contains Jupyter notebooks for backtesting a strategy for buying the dip. The goal is to evaluate the profitability of such a strategy.

## Features

- Implementation of the trading strategy logic
- Performance evaluation using key metrics
- Data handling and preprocessing for backtesting  
- Visualization of backtest results  

## Getting Started

1. **Clone the repository:**

    ```bash
    git clone https://github.com/tradingsteady/Buy-the-Dip.git
    ```

2. **Open the Jupyter notebooks:**

    ```bash
    jupyter notebook
    ```
